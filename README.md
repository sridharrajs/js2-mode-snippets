## Ya-Snippets for js2-mode

### Functions

`fn` is the short form for `function`. So,

* Named function => `nfn`
* Anonymous function => `afn`
* Request function => `rfn`
* Static function => `sfn`

`nfn ` gives

    function <Tab> (<Tab>) {
      <Tab>
    }

### Lodash

`_` is the short form. So,

* `_.each()`   => `_e`  
* `_.map()`    => `_m`  
* `_.reduce()` => `_r`
